https://git-scm.com/download/win

git init
создаёт локальный репозиторий. В текущей папке появится подпапка с именем .git (по умолчанию она скрыта)

git add main.cpp
добавить файл к списку, для отправки в репозиторий

git commit -m "save main.cpp"
сохранение изменений с комментарием

git branch -M main
обновление ветки проекта

git status
проверка состояния файлов


Удаленный репазиторий уже создан, его и используем.

git remote add origin https://github.com/Maxim-2005/Synergy.git
сопоставление локального репозитория с удаленным

git push -u origin main
синхронизация локального репозитория с удаленным
