Plugin for managing litters and pets with automatic data synchronization.

== Description ==

Pets Nursery is a WordPress plugin for managing animal litters and individual pets. It provides:

* Custom post types for Litters and Pets
* Automatic synchronization of birth dates
* Parent-child relationships
* Search and filtering capabilities
* Multi-language support (English, Russian)

== Features ==

* Create and manage litters with birth dates
* Add pets to litters with automatic data sync
* Search pets by nickname
* Automatic title generation from pet nicknames
* Responsive design
* Translation ready

== Installation ==

1. Upload the `pets-nursery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating Litters and Pets in the admin menu

== Frequently Asked Questions ==

= How do I add a pet to a litter? =

When editing a litter, use the search field to find existing pets or create new ones directly from the litter page.

= Can I change a pet's birth date manually? =

Yes, but if the pet is linked to a litter, the birth date will be synchronized with the litter's birth date.
