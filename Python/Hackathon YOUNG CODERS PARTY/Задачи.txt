№ 1
У Васи есть N рублей купюрами. Он хочет разменять их на монеты по 1, 3 и 5 рублей.
Автомат, который это делает, принимает на вход купюры, а выдает монеты, причем наименьшее их количество.
Напишите программу, которая на входе получает одно число — количество рублей, и выдает одно число на выходе — минимальное число монет данного номинала.
Входные данные: количество денег
Выходные данные: минимальное количество монет
Пример входных данных: 14
Пример выходных данных: 4

№ 2
Книга состоит из N страниц. Известно, что для записи всех номеров страниц этой книги было напечатано M символов.
Напишите программу, которая вычисляет N из M.
Входные данные: число символов для записи страниц книги
Выходные данные: количество страниц в книге
Пример входных данных: 13
Пример выходных данных: 11

№ 3
Последовательность Фибоначи задается следующим образом: каждый следующий элемент является суммой двух предыдущих.
Начало выглядит так: 1, 1, 2, 3, 5, 8, 13, 21, 34, ... Напишите программу, которая выводит n-ый элемент этой последовательности.
Входные данные: n - номер числа последовательности
Выходные данные: F(n) - n-ое число из последовательности
Пример входных данных: 10
Пример выходных данных: 89

№ 4
Дана строка, в которой через пробел написаны натуральные числа. Напишите программу, которая выводит наибольшее произведение трех чисел их данного ряда.
Входные данные: строка чисел через пробел
Выходные данные: наибольшее произведение из трех чисел
Пример входных данных: 1 4 3 2
Пример выходных данных: 24

№ 5
Факториалом числа n называют последовательное произведение всех чисел от 1 до n и обозначают n!. Например, 5! = 1 * 2 * 3 * 4 * 5 = 120.
Напишите программу, которая из n! получает n.
Входные данные: n! - факториал числа n
Выходные данные: n - исходное число
Пример входных данных: 120
Пример выходных данных: 5

№ 6
В корпоративный компьютер проник вирус, который удалил некоторые пробелы, добавил некоторые лишние символы в документе, где хранились данные по поставкам микроволновых печей. В документе использовались первые три буквы от наименований производителей. Чтобы как-то восстановить документ, необходимо написать функцию, которая возвратит повторяющиеся группы из трех знаков идущих подряд. Если нет повторов, то выводится 0
Пример: "anb" и "abn" разные группы, "ans" и "ans" одна группа.
Входные данные: строка
Пример входных данных: "ghjghjghjghjghj"
Выходные данные: строки, где выведены группы повторяющихся букв,за ними написаны числа, указывающие сколько раз эти группы повторяются.   Строки должны быть отсортированы в лексикографическом порядке.
Пример выходных данных: "ghj,hjg,jgh,5,4,4"

№ 7
Напишите функцию, которая будет принимать массив из целых, положительных чисел, и проверять, можно ли перемножить 3 из этих цифр и получить такое число, чтобы 2160 было кратно ему.
Входные данные: строка состоящая из целых, положительных чисел
Пример входных данных: 3,1,6,3,7,7,8,2,3
Выходные данные: 0 или 1. Если да, то 1. Если нет, то 0.

№ 8
Фишинговые сайты используют url-адреса, схожие с настоящими, но при этом в них заменена одна буква.
Ниже написаны буквы которые могут быть заменены друг на друга: “w” ⟷ “v”, “s” ⟷ “c”, “d”⟷"t"
Вводится название сайта, ваша задача вывести всевозможные его мошеннические аналоги.
Входные данные: название сайта
Выходные данные: всевозможные названия фишинговых сайтов через запятую, отсортированные в лексикографическом порядке
Если букв для замен нет, выведите -1
Пример входных данных: codenrock
Пример выходных данных: codenrosk,cotenrock,sodenrock

№ 9
Разошлите сообщения клиентам банка следующего содержания:
Уважаемый(-ая) Иван Иванов на вашем счете (карта VISA4231) осталось 400 руб.
Входные данные: Имя Фамилия, номер карты, баланс, пол(М или Ж), через запятую, без пробелов
Пример: Дарья Гонева,5110000134567579,15000,Ж
Выходные данные: сообщение по образцу
Пример: Уважаемая Дарья Гонева на вашем счете (карта VISA7579) осталось 15000 руб.

№ 10
Клиенты компании, место работы которых находится на расстоянии R от офиса, должны получить СМС-оповещение.
Выведите получит ли СМС наш клиент.
Входные данные:  координаты офиса (2 числа),  расстояние R, координаты места работы (2 числа)
Пример входных данных: 10,12,3,11,12
Выходные данные: 1 если получит СМС, 0 если нет
